# Security Policy

## Supported Versions

We primarily support the latest version of VTOP 2.0. Security patches will be applied to the `main` branch.

| Version | Supported |
| ------- | --------- |
| 2.0.x   | ✅ Yes    |
| < 2.0   | ❌ No     |

## Reporting a Vulnerability

We take the security of our project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately to our team.

### How to Report

Please email [insert security email address] with the following information:

- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.
- Any suggested fixes.

We will acknowledge your report within 48 hours and provide a timeline for a fix. We appreciate your help in keeping VTOP 2.0 secure!
